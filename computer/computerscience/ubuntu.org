** 时区
   Linux设置方法
   首先安装NTP：yum install ntp
   修改/etc/ntp.conf如下：
   
   server ntp1.aliyun.com prefer
   server ntp2.aliyun.com
   手工发起同步：ntpdate ntp1.aliyun.com
   启动NTP服务：service ntpd start
   设置开机启动：chkconfig ntpd on
   检查设置：chkconfig --list ntpd
