* Angular 
** 简介 
   网页 app 的 *组件* 化 
   #+begin_quote
   本来一个对象是分三层了 : HTML 构建对象内容，CSS 构建对象的外观，JAVASCRIPT 构建对象的行为。现在都放一个文件里面了。
   #+end_quote
   
   
** 安装使用 
   #+begin_src sh -n
     npm install -g @angular/cli
     # 创建项目
     ng new my-app
     # 启动服务程序
     ng serve --open
     ng serve -o
   #+end_src
