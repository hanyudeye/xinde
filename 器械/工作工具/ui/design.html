2、设计创意参考

  • Dribbble - Show and tell for designers 追波，不多说。

  • DeviantArt - The largest online art gallery and community 也很有名，不多说了。
  • https://www.pinterest.com/ 素材创意大集合，非常棒的网站，也很有名，不赘述了。

  • 站酷 (ZCOOL) 国内设计比较有名的网站，不多说。
  • UI中国 虽然注册用户量和知名度不如站酷，但是专攻UI领域，感觉还不错。

  • Website Templates themeforest 知名度非常高的模版销售网站，从上面可以看到很多目前国外流行的页面风格、动效，创意也非常不错。
  • WordPress Themes Loved By Over 317k Customers elegantthemes 国外很有名的wordpress模版设计团队，创意十足，不过可惜的是最近不思进取，一直没有太大创新，还在售卖一年多前就发布的模版，不过他们的blog可以看看，对wordpress开发有很大帮助。
  • Home - YOOtheme 也是非常棒的国外网站模版设计公司，包括wordpress和joomla!，设计简洁有想法，还有他们推出的前端框架 UIkit（类似Bootstrap），有兴趣的朋友可以了解一下。
  • Reeoo - web design inspiration and website gallery 很棒的网页设计参考网站，可以按照颜色筛选，更新勤快，创意十足。

  • http://www.materialup.com/ Material Design相关设计作品的参考网站。
  • 100 Days UI - FREEMIUM RESOURCES 一个Dribbble牛人的每日一设作品，有源文件下载和HTML代码实现哦！绝壁学习极品。

  • https://webgradients.com/ 轻量的渐变配色网站，配色很有逼格。
  • Color Tool - Material Design Material Design官方的配色工具，可以通过取色快速查看界面整体效果。
  • Sketchfab - Your 3D content on web, mobile, AR, and VR. Sketchfab-全球知名的 3D 设计模型在线展示平台。


3、前端／动效／用户体验

  • Codrops | Useful resources and inspiration for creative minds 由于鄙人是从前端开发转的UI设计，所以一直对前端各种效果都情有独钟，其中Codrops就是我几年以来基本上隔三差五就会去逛的网站，创意十足，动效酷炫，每周集锦很有意思，强烈推荐，关键在于所有动效都是可以免费下载使用的！
  • http://codyhouse.co/ 也是非常棒的一个前端动效网站，Codrops经常会收录它的一些好作品。
  • Tutorialzine 同样有很多好玩的前端效果和知识，但是更新并不勤快，可以偶尔看看。
  • Trending CSS repositories on GitHub today · GitHub github就不用多说了吧？我会经常到trending里筛选CSS/HTML/Javascript的语言分类来看看是否有最新的好玩的前端作品，这里应该算是全球开源代码最领先的地方了。
  • GoodUI 通俗易懂的UI设计准则，很有意思。
  • Medium 里面有很多很棒的设计类文章，值得参考。
  • CodePen 很棒的前端网站，可以直接预览前端效果和使用代码。
  • Font Awesome, the iconic font and CSS toolkit 最早的开源免费字体图标库，以前用Bootstrap+FontAwesome基本上就能搭建一个像模像样的网站，图标种类繁多，使用方便。
  • Wappalyzer 可以查看当前网页的前端技术使用情况，非常方便。


不管有多好的设计素材网站，最重要的还是自己多看、多思考、多练习，这样才能不断进步，与君共勉。

4、工具类

  • https://tinypng.com/ 在线压缩PNG/JPG图片网站，能够大幅度减小图片尤其是PNG图片的size，方便好用。
  • Icon Font &amp;amp;amp;amp;amp;amp;amp;amp; SVG Icon Sets â IcoMoon 字体图标管理／生成网站，功能强大，国内的iconfont应该是参考它做的。
  • Iconfont-阿里巴巴矢量图标库 icomoon虽然很棒，但是国内访问速度还是成问题，所以现在还是比较常用iconfont，也挺好用的。
  • GuideGuide Photoshop必装的参考线插件神器，谁用谁知道。
  • RightFont - The future of font manager software for Mac OS X 非常棒的字体管理软件，能够快速加载字体，并且快速在设计稿中引入矢量图标，个人感觉比在设计稿中使用iconfont方便。
  • Sketch - Professional Digital Design for Mac Sketch，Mac下的UI设计神器，制作UI界面快速简单，而且正版也不贵，缺点是没有Windows版本。
  • https://github.com/utom/sketch-measure Sketch measure，Sketch中必装的插件，能够导出设计稿的HTML文件供前端实时查看，免去了切图的烦恼，还能切换设备分辨率，非常方便。
  • https://www.flinto.com/ 轻量级的交互软件，能够与Sketch一起使用，简单方便。
  • Principle - Animated Design 功能强大的交互设计软件，用它能够做出很不错的app动效。
  • Framer - Design tool for creating interactive designs, interfaces and animations 能够直接用代码做交互／动效，个人感觉很不错，广受国外一线设计师的好评。
  • Craft by InVision LABS Invision团队推出的PS／Sketch插件，能够快速添加图片／文字等功能。
  • https://g2.alipay.com/ G2-蚂蚁金服团队的图表组件工具，设计图表的时候可以参考。
  • GitBook 使用Markdown编写文档，采用Git管理方式，非常方便高效。
  • Device metrics - Material Design 谷歌设计团队推出的设备相关参数参考页面，很完整。
  • ECharts 百度团队的纯 Javascript 图表库，算是比较成熟的解决方案。
  • 常见设计规范与 Sketch 源文件下载集合 - 知乎专栏 这篇文章收集了相对完整的设计规范及组件库下载资源。
